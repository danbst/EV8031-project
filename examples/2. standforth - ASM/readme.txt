StandForth - форт-інтерпретатор для EV8031/AVR

Форт - інтерпретатор, операційна система, мова програмування, яка відносно легко реалізовується на асемблері, навіть простіше ніж сам асемблер.

StandForth було розроблено в рамках бакалаврської роботи. Важливо відмітити, що StandForth знаходиться далеко від завершення. У ньому працює тільки невелика множина слів з оригінального Форту (особливо неприємною особливістю є нереалізовані слова IF THEN ELSE LOOP WHILE). Також код містить баги, які відловити поки-що не вдалось (і, скоріш за все, ніколи не вдастся).

Для чого потрібен даний код?
1. Приклади використання асемблених команд. Інтенсивна робота з пам'яттю.
2. Приклад реалізації Форта на гарвардській архітектурі
3. Приклад організації передачі даних через UART
4. Приклад, чому краще писати на С, а не на асемблері.

У каталозі розміщено не усі файли проекту. В оригіналі, було розроблено Python-макропрепроцесор, котрий виконував інтелектуальні макрозаміни з оригінального standforth.asm і генерував на їх основі _gen_standforth.asm. Код усіх інших файлів проекту можна глянути на GitHub: ...

Файли:
- _gen_standforth.asm - асемблерний код Форта
- ev8031.asm - розміщення адрес пристроїв в оперативній пам'яті
